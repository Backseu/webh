There are many ways to use scripts in games. Usually, only part of the game is scripted, and the game client uses the script to refer to
or invoke the script for particle effects or animation. A common scripting language for game development is Lua Script. There is also
a development tool with an internal scripting language, typically Flash or Unreal Engine that uses Action Script.

    1. C#(script)

    (C# download)

    A component programming language that treats everything as an object, pronounced C-sharp. In June 2000, Microsoft developed the .NET platform.
    It is based on C ++ (City Plus Plus), similar to Visual Basic or Java. Therefore, it has advantages such as Visual Basic, Java and C ++. It reflects a component-based software development paradigm that has all the user-friendliness of visual language, object orientation of C ++, and multiplicity suitable for Java's distributed environment processing.

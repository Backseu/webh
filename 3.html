    Web sites that we connect to on a daily basis largely consist of three components. 'Hyper Text Markup Language (HTML)', 'Cascading Style Sheets' (CSS), and 'Javascript'. HTML provides a large skeleton of web pages, and CSS manages design elements such as color and font.
